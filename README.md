# Wordle Game (React + TypeScript)

Live Demo — [Ознакомиться](https://bobicentr.github.io/wordle_react_ts/)

## О проекте

Wordle Game — это реализация игры Wordle на стеке **React + TypeScript + TailwindCSS v4** с упором на корректную алгоритмику и синхронизацию интерфейса с логикой состояния. Проект не является простым клоном: реализована точная обработка повторяющихся букв, управляемые анимации и строго типизированная игровая модель.

Приложение демонстрирует работу с алгоритмами проверки, иммутабельностью состояния и продвинутыми UI-эффектами.

## Технологии

- React  
- TypeScript  
- TailwindCSS v4  
- Vite  

## Как установить локально

Склонировать репозиторий  
```bash
git clone https://github.com/bobicentr/react_wordle_ts.git  
cd react_wordle_ts  
```

Установить зависимости  
```bash
npm install  
```

Запустить проект  
```bash
npm run dev  
```

Открыть ссылку из терминала (обычно [http://localhost:5173](http://localhost:5173))

## Ключевые функции

- Проверка слов с корректной обработкой повторяющихся букв  
- Двухпроходный алгоритм подсветки результатов  
- Валидация слов по словарю  
- Полный ввод с клавиатуры (включая Backspace и Enter)  
- Блокировка ввода после завершения игры  
- Мгновенный сброс состояния  
- Анимации раскрытия, shake и pop-эффекты  

## Архитектура и UX

- Строгие типы для статусов ячеек и состояния игры  
- Иммутабельные обновления сетки  
- Чистые функции проверки без побочных эффектов  
- Глобальный keyboard listener с корректным cleanup  
- 3D flip анимации с задержкой по индексу ячейки (staggered effect)  
- Смена цвета синхронизирована с серединой поворота
